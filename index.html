Installing on macOS

Nexacraft is currently unsigned (no Apple Developer ID), so macOS will block it the first time you open it. This is normal for indie alpha builds. Follow these steps:

1
Open the .dmg and drag Nexacraft into your Applications folder. (Don't try to run it from the .dmg window — it won't work.)
2
Open Terminal (⌘ + Space → type "Terminal") and paste: 
xattr -dr com.apple.quarantine /Applications/Nexacraft.app
Press Enter. (No output means success.)
3
Now double-click Nexacraft in your Applications folder. It'll open normally.
Why is this needed? / Common error messages

macOS adds a "quarantine" flag to every file downloaded from the internet. For unsigned apps (no $99/yr Apple Developer ID), macOS shows one of these errors:

  • "Nexacraft is damaged and can't be opened" — the app is fine; macOS just blocks it. The Terminal command above fixes it.
  • "Nexacraft can't be opened because Apple cannot check it for malicious software" — same root cause. Same fix.

The xattr command removes the quarantine flag so macOS stops blocking the app. It's safe — it only affects this one file.

About Nexacraft

Nexacraft is a voxel sandbox game inspired by the classics. Build, explore, and survive across procedurally generated worlds — overworld biomes, mob-filled caves, dungeons, villages, and even the Nether.

  • Infinite procedurally-generated worlds
  • Survival, Creative, and Spectator game modes
  • Mobs, combat, weather, day/night cycles
  • Crafting, mining, building, sleeping in beds
  • Villages, dungeons, and the Nether dimension
